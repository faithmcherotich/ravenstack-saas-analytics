# 📊 RavenStack SaaS Churn & Revenue Analysis

## Overview
This project analyzes customer churn behavior and revenue trends for a SaaS company. The goal is to identify key drivers of churn, quantify revenue at risk, and uncover insights to improve retention.

---

## Business Questions
- What is the churn rate across plan tiers?
- What are the main drivers of churn?
- How much revenue is at risk from inactive users?
- Which features drive customer retention?
- How does retention evolve over time?

---

## Tools Used
- SQL — data extraction and aggregation  
- Python — analysis and visualization  
- Power BI — dashboarding  

---

## Key Insights
- Churn is consistent (~22%) across all plan tiers, indicating pricing is not the main driver  
- Feature limitations are the leading cause of churn  
- Enterprise customers contribute the highest revenue at risk  
- Retained users show higher engagement with key features  
- Revenue increased significantly despite stable churn
